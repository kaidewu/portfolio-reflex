# Kaide's Portfolio

A portfolio build it with [Reflex](https://reflex.dev)

# Getting Started
### Prerequisites:
Download and install Python (Version 3.10+ is recommended).

### Setting up the project:
1. Clone the GitHub repository:
```
git clone https://github.com/kaidewu/portfolio-reflex
```
2. Navigate to the project directory:
```
cd portfolio-reflex
```
3. (Recommended) Create a Python virtual environment: You can follow the Python official documentation for virtual environments.
```
python3 -m venv venv
```
4. Activate the virtual environment:
  - On Windows:
    ```
    .\venv\Scripts\activate
    ```
  - On MacOS and Linux:
    ```
    source venv/bin/activate
    ```
6. Install the required Python packages from requirements.txt:
```
pip install -r requierements.txt
```
8. Start Reflex Application:
```
reflex init
```
10. Run Reflex Application:
```
reflex run
```
